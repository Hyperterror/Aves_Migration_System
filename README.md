# Bird Migration Prediction System
## 📊 Dataset

- 4.8 million GPS records  
- 22 birds tracked  
- Converted into migration events  

## ⚙️ Pipeline

- Data preprocessing  
- Migration detection  
- Feature engineering  
- Model training  

## 🤖 Models Used

- Logistic Regression  
- KNN  
- SVM  
- Decision Tree  
- Random Forest  
- XGBoost
