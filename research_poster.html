🔎 Problem Statement
Accurately predicting bird migration success from GPS telemetry is critical for conservation. Raw sensor data suffers from severe class imbalance and a single dominant individual — preventing reliable machine learning.
💡 Motivation
  • Great Egrets are indicator species of wetland health.
  • Migration failure signals habitat loss or tag failure.
  • ML models can automate large-scale telemetry monitoring.
  • SMOTE + bias correction can rescue imbalanced wildlife datasets.
📊 Dataset
  • Source: Audubon Canyon Ranch (ACR) Telemetry Project
  • Raw GPS fixes: ~4.8 million @ 30-min intervals
  • Individuals: 22 tagged egrets (GREG_1 – GREG_22)
  • Raw columns: 48 sensor channels

⚠️ Bias Issues Identified
  • 18:1 class imbalance (Success vs Failed)
  • All 6 test samples were class-1 (trivial accuracy)
  • GREG_5 dominated: 15/29 events (52%) from one bird
  • 100 km home-return threshold too generous → mislabeled events
🔑 Key Insights
  • Duration & displacement are top predictors.
  • KNN & Random Forest achieved highest F1 (0.972).
  • Bias correction improved minority-class recall significantly.
  • Stratified CV revealed true per-model variance.

✅ Conclusion
KNN & Random Forest achieved the highest F1-Score (0.972) after bias correction, confirming that ensemble/proximity-based methods handle small, imbalanced telemetry datasets best. A 4-layer bias correction pipeline (strict labeling + SMOTE + per-bird weighting + Stratified CV) transformed a naïve 100% accuracy (trivial) result into honest, generalizable performance, reducing the class-imbalance ratio from 18:1 to 11.7:1 and exposing real model differences.

🧬 Feature Engineering Buckets
  • Geospatial: displacement, path length, tortuosity, bearing
  • Temporal: duration, start/end month, daily distance
  • Movement: speed stats, heading consistency, fix rate
  • Sensor: temperature, battery voltage & depletion rate
  • Altitude: mean, max, min, range, std
  • Behaviour: rest stops, movement ratio, long-flight segments
